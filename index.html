Code example

Task from the basics of JavaScript Hixlet: Write a makeItFunny() function that takes a string as input and returns a copy of it with every nth element converted to uppercase. n– is set at the input to the function:

                            const makeItFunny = (str, n)=>{
                                let result = '';
                                let i = 0;
                              while (i <str.length) {
                                if ((i + 1) % n === 0 && (i + 1)>= n){
                                  result += str[i].toUpperCase(); 
                              } else {
                                  result += str[i];
                                }
                              ++i;
                              }
                              return result
                              }

                              export default makeItFunny;
